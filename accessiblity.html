Accessibility

Accessibility is a necessity when creating a professional and functioning website. Having your website be accessable means that no matter a person's disability they should be able to navigate and use your website as well as any able-bodied person would. This includes but not limited to: making sure the website works well when using s screen reader, using colors that colorblind people can understand, and making use of certain attibutes to your tags so people with low motor skills can still navigate your website.

Below you will see more indepth ways to make your website accessible to all.

Use Proper Alt Tags on All of Your Images

For those who are vision impaired and have to rely on a screen reader to navigate the website, alt tags are a must.
They are what are read by the screen reader to the user. So make sure to use descriptive language in your alt, so people will understand what the image is.
Also when the image fails to load the alt is what is displayed instead. So there is no reason not to use it.

Give Your Links Descriptive Names

For those who rely on screen readers, links are seperate from the rest of the page.
Without the context of surrounding words, links that just say, "Click me" have no meaning to those that use screen readers.
Using short but descriptive words such as, "About Us" or "Studies on Women's Health", go a long way in terms of accessibilty.

Make Sure The Website Can Be Navigated With a Keyboard.

Best way to check to see if your website can be navigated with your keyboard is to simply go to your website and start pressing tab.
If you can't access parts of the website, neither can people with low motor skills. To make the website functionable with a keyboard use the tabindex tag.
Setting it to -1, makes it impossible to access with the tab key. Using 0 puts it in order of how the source code flows.
And using positive numbers puts the tab order as you set it. The average user will most likely never notice it. But those who need it will be greatful for it.
